JetBrains 학생 인증 방법

  1. 사이트 접속!


    2. JetBrains 학생 라이센스 인증 사이트

    위 사이트에 접속합니다.

  2. GitHub Student Developer Pack 링크 접속

    3. 무료 교육 라이센스를 받을 수 있는 여러 옵션들이 나와있습니다.

    저는 github로 인증하는 방법을 설명하겠습니다.

    GitHub Student Developer Pack 링크를 클릭합니다.

  3. Get your pack 버튼 클릭

    4. 만약 로그인 창이 뜨면 로그인 하면 됩니다.

  4. Get student benefits 버튼 클릭

    5. 학생 인증하면 받을 수 있는 혜택들이 나와있습니다.

    Get student benefits 버튼을 클릭해 github에서 인증을 시작합니다.

  5. Add an email address 버튼 클릭

    6. 학교 이메일 계정을 추가할 것입니다.

  6. 학교 이메일 추가

    7. Email address칸에 학교 이메일을 입력합니다.(이메일 형식은 ...ac.kr 입니다.)

    학교 이메일 주소를 입력하고 Add를 클릭합니다.

  7. 학교 이메일 확인

    8. 메일이 오면 Verify email address를 클릭하고 github에 다시 돌아갑니다.

  8. github 사용 플랜 입력하기

    9. How do you plan to use Github? 칸에 github 사용 플랜 입력합니다.

    Submit your information 버튼을 클릭해서 제출하면 학생 인증이 완료됩니다.

  9. 끝!

    10. 학교 이메일 인증이 완료되면 이렇게 학교 인증 뱃지를 얻게됩니다.
